## 介紹

因為我朋友當家教有此需求，所以我想使用Python來封裝並完成這個專案。

## 特色

- 透過Google表單添加問題和答案
- 自動匯入Google sheet
- 利用Python提取Google sheet儲存格內容
- 可依照需求提取出相應的題目，並且隨機處理
- 輸出成Word docx格式
- 能夠過OpenCV等OCR自動掃描並輸入題目 `未來目標`
